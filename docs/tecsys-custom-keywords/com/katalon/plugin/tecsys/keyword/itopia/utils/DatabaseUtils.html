        • java.lang.Object prepareWhereClauseFromMap(java.util.Map map, java.lang.Integer endIndex = 0)

          The purpose of this method is to copy a record from the table provided in the first argument and create another record from it. If the intended record to create already exists in the table, it will not try to create it. This method does not return anything, however, it performs operations on the database. It takes the following arguments: 1. First Argument is the table name to copy record from 2. Second Argument is the list of map. The map represents the column name and its intended value in key value pairs. The record created will have these values 3. Third Argument is optional is the list of columns to remove. This is usually to prevent copying a column with unique or self-generating values (like ID) 4. Fourth Argument is also optional. One can define how much elements in second argument he can take from the beginning. This will enable user to have more control over the filters he can apply to create a new record.
